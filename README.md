System Overview

This project uses the Arduino Nano 33 IoT to construct a basic smart lighting system for an assisted living setting. The technology is intended to assist Linda, an elderly resident, in entering her house securely at night. Linda activates the porch light for thirty seconds and the hallway light for sixty seconds by pressing a push button at the door. This gives her enough light to open the door and enter the house securely.

The device employs a push button to initiate the lighting sequence and two LEDs to mimic the porch and hallway lights. This solution uses Arduino to illustrate fundamental ideas of modular programming and embedded system control.
